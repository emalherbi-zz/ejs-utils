Com esse plugin é possível realizar

CPF/CNPJ

- Validar CNPJ e CPF

Dinheiro

- Transformar valores em US ou BR

Digitos

- Remover máscara e pegar somente digitos do campo

Máscara

- Criar máscara para os campos de: DATA, HORA, TELEFONE, CEP, CPF, CNPJ, NUMEROS, VALOR, VALORBR

Input

- Somente Leitura ou Editável

Zeros

- Adicionar zeros a esquerda de um valor

Formulário

- Limpar todos os campos do formulário

Tabela

- Remover Tabela
- Totalizar a tabela (formatos US e BR)

Select

- Pegar o valor do Atributo do Item Selecionado
- Remover a opção selecionada do Select
- Remover todas opções do Select

Alert

- Se não validou o retorno é um Alert e false // se não true

Como Utilizar

CPF/CNPJ

: Retorno é boleano

CPF.valida(cpf)


CNPJ.valida(cnpj)

Dinheiro

: Retorno é o valor formatado

Money.formatBr(vlr)
Money.formatUs(vlr)

Digitos

: Retorno somente digitos sem máscara

Dig.only(vlr)
