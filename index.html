Com esse plugin é possível realizar

CPF/CNPJ

- Validar CNPJ e CPF

Dinheiro

- Transformar valores em US ou BR

Digitos

- Remover máscara e pegar somente digitos do campo

Máscara

- Criar máscara para os campos de: DATA, HORA, TELEFONE, CEP, CPF, CNPJ, NUMEROS, VALOR, VALORBR

Input

- Somente Leitura ou Editável

Zeros

- Adicionar zeros a esquerda de um valor

Formulário

- Limpar todos os campos do formulário

Tabela

- Remover Tabela
- Totalizar a tabela (formatos US e BR)

Select

- Pegar o valor do Atributo do Item Selecionado
- Remover a opção selecionada do Select
- Remover todas opções do Select

Alert

- Se não validou o retorno é um Alert e false // se não true

Como Utilizar

CPF/CNPJ

- Retorno é boleano 
CPF.valida(cpf)CNPJ.valida(cnpj)

Dinheiro

- Retorno é o valor formatado 
Money.formatBr(vlr)Money.formatUs(vlr)

Digitos

- Retorno somente digitos sem máscara 
Dig.only(vlr)

Máscara

- Retorno é o campo com máscara 
Mask.mascara(this, Mask.mdata)
Mask.mascara(this, Mask.mhora)
Mask.mascara(this, Mask.mtelefone)
Mask.mascara(this, Mask.mcep)
Mask.mascara(this, Mask.mcpf)
Mask.mascara(this, Mask.mcnpj)
Mask.mascara(this, Mask.mnumeros)
Mask.mascara(this, Mask.mvalor)
Mask.mascara(this, Mask.mvalorbr)

Input

- Retorno é o campo Editável ou Somente a Leitura 
Input.readOnly(id)
Input.enabledField(id)

Zeros

- Retorno é o campo formatado com zeros a esquerda 
Zeros.left(str, max)

Formulário

- Retorno é o formulário limpo 
Form.clear(idform)

Tabela

- Retorno é a remoção da tabela 
Form.remove(idtable)
- Retorno é o valor calculado dos campos da tabela, pode ser inteiro ou real 
Form.calculate(idtable, posfield)
Form.calculateBr(idtable, posfield)

Select

- Retorno é o valor do atributo selecionado 
Select.option_select_attr(idselect, attr)
- Retorno é a opção removida 
Select.remove_selected_option(idselect)
- Retorno é todas opções removidas 
Select.remove_all_option(idselect)

Alert

- Se não validou o retorno é um Alert e false se não true 
Alert.show(id, msg)
