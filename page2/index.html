The Project

The Javascript Utils!

With this plugin is possible to do:

  • Table Calculate Us/Br - Calculate Us/Br values from table.
  • Trim - Strip whitespace from the beginning and end of a string.
  • Zeros - Add zeros to the left of a string number.
  • Validate Field - Validate the field and displays a warning message if empty.
  • Scroll to ID - Go To ID of Body With Animate.
  • Body Top - Go To the Top of Body With Animate.
  • Timeout - set html Timeout .
  • Select - return a specific attribute, remove an option from select, remove all options from select .
  • Radio - return radio checked from form.
  • Money Format - format a number in Brazil or US.
  • Mask - mask for input fields.
  • Input - input readonly , enabledField.
  • Form Clear - Clear all input's form.
  • Div Position - positions in specific div hide all div.
  • Data Format - returns Date format Br or Sql and time now.
  • Dig - returns only the input digits.
  • CPF Validate - return CPF Validate.
  • CheckBox checked - return CheckBox checked.
  • CNPJ Validate - returns CNPJ Validate .

Usage

<!-- Latest compiled and minified JavaScript -->
<script src="http://emalherbi.github.io/ejs-utils//dist/ejs-utils.js"></script>
